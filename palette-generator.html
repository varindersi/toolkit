How to Use the Palette Generator

  1. Click the "Generate Palette" button to create a random color palette
  2. Adjust the number of colors using the slider (3-10 colors)
  3. Choose your preferred color format (HEX, RGB, or HSL)
  4. Click on any color to copy its value to your clipboard
  5. Use the "Export as Image" button to save your palette as a PNG file
